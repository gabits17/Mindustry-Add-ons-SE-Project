### User story

As an EXPERIENCED USER, I want to use a reliable method to track and modify the edits I make to the map, so that I can efficiently undo or redo any mistakes while building.

#### Notes

Undo and redo actions

### Review
Diogo Antunes (67763)

I can see this as a feature similar to snipping tool. It might be interesting to consider, in example, only deleting pipes and ducts in a certain mode to avoid destroying buildings when needing to reorganise pipelines.

Dinis Neves (68130)

There will be a challenge in the need to access the blocks copied / cut and the store them to be used later, pasted.
It is a functionality that the game lacks, moving by copying and cutting would be a welcomed (quality of life) feature that does not yet exist in the game.
